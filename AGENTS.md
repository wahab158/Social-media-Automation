# Project Context: Social Media Automation

This project is a multi-tenant social media automation pipeline.

## Tech Stack
- **Backend**: FastAPI (Python)
- **Frontend**: React (Vite)
- **Database**: SQLite + Google Sheets
- **Media**: Cloudinary
